== Description ==

View demo - [Side Slide Menu Demo](http://workshop.systems/sideslide/) 

SideSlide Menu is a lightweight responsive WordPress side menu. It utilizes CSS3 transitions and Javascript (without jQuery) on frontend which guarantees good performance on lower - end mobile devices, while on better mobile devices and PC's it is smooth as silk. All colors are customizable, so you can match your website visual identity perfectly.

Side Side Menu also comes with 3 shortcodes that let you hide/show/toggle menu from anywhere in your theme.

Features

* Customize any color
* Lightweight
* 350+ FontAwesome Icons
* 600+ Google Fonts
* Supports touch gestures on touch enabled devices
* 3 shortcodes that let you manually show/hide/toggle menu
* You can insert custom logo
* Place for additional information below menu
* Intuitve Admin Panel

== Installation ==

1. Upload `sideslidemenu.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are done! The plugin settings are under Settings > Side Slide Menu
